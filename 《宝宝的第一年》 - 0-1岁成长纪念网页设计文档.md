## 📚 项目概述
基于Microsoft Edge年度回顾的优美设计风格，为新生儿的家长创建一个**沉浸式、时间轴型的成长纪念网站**。通过精心的动画、配色和交互设计，记录宝宝从0个月到12个月的每一个重要时刻，让家长以优雅而富有情感的方式回顾和分享宝宝的成长故事。

---

## 🎨 设计理念与风格指引
### 1. 设计核心价值
+ **温暖感**：采用柔和、温暖的色彩系统，传递家庭、爱、陪伴的情感
+ **渐进式叙述**：每个月份独立成章，但整体形成连贯的成长叙事
+ **沉浸式体验**：通过动画和视觉层次，让家长完全沉浸在回忆中
+ **易分享性**：整个成长历程可轻松分享给家族成员和朋友

### 2. 视觉美学参考
继承Microsoft Edge年度回顾的特点：

+ ✅ 渐变背景随内容变化
+ ✅ 层次丰富的装饰图片布局
+ ✅ 柔和的卡片设计与阴影
+ ✅ 平滑的过渡动画
+ ✅ 清晰的进度指示
+ ✅ 引导性的交互提示

### 渐变方向与深度
每个背景采用**45度斜向渐变**，从上方的亮色逐渐过渡到下方的略深色：

```css
background: linear-gradient(135deg, 
  rgba(255, 192, 203, 0.9) 0%,   /* 浅粉红 */
  rgba(255, 240, 245, 1) 100%);  /* 乳白色 */
```

**优点**：

+ 创造纵深感和空间感
+ 提供足够的背景对比度用于文字可读性
+ 柔和不刺眼，适合长时间观看

## 🎬 动画与交互设计规范
### 1. 页面切换动画
**进入新月份页面**（点击"下一个月"时）：

```javascript
// 动画序列
1. 背景色渐变过渡   (400ms, ease-in-out)
   旧色系 → 新色系

2. 卡片内容淡出     (300ms, ease-out)
   opacity: 1 → 0

3. 新卡片内容淡入   (300ms, ease-in)
   opacity: 0 → 1
   同时伴随轻微缩放: scale(0.95) → scale(1)

4. 装饰图片过渡     (500ms, ease-out)
   旧图片淡出 → 新图片淡入

5. 进度指示更新     (400ms, ease-out)
   数字计数动画: oldProgress → newProgress
```

**缓动函数推荐**：

+ 背景过渡：`cubic-bezier(0.25, 0.46, 0.45, 0.94)` （标准缓动）
+ 内容淡出：`cubic-bezier(0.4, 0, 1, 1)` （快速淡出）
+ 内容淡入：`cubic-bezier(0, 0, 0.2, 1)` （缓入）
+ 弹性效果：`cubic-bezier(0.34, 1.56, 0.64, 1)` （轻微弹跳）

### 2. 装饰图片的视差效果
在浏览器支持的情况下，使用CSS或JavaScript实现：

```css
/* 左侧图片向上移动略快 */
.deco-left {
  transform: translateY(calc(var(--scroll-y) * 0.3px));
  transition: transform 0.3s ease-out;
}

/* 右侧图片向上移动略慢 */
.deco-right {
  transform: translateY(calc(var(--scroll-y) * 0.2px));
  transition: transform 0.3s ease-out;
}
```

这样当用户滚动时，不同图片以不同速度移动，创造深度感。

### 3. 卡片悬停效果
当鼠标悬停在卡片上时：

```css
.content-card:hover {
  /* 阴影增强，表示浮起 */
  box-shadow: 0 16px 48px rgba(0, 0, 0, 0.16);
  
  /* 轻微放大 */
  transform: scale(1.02);
  
  /* 过渡时间 */
  transition: all 0.3s cubic-bezier(0.34, 1.56, 0.64, 1);
}
```

### 4. 按钮交互反馈
**三态设计**：

1. **普通状态**：
    - 背景：浅色
    - 文字：深色
    - 阴影：轻微
    - 无变换
2. **悬停状态**（:hover）：
    - 背景：加深5%
    - 放大：scale(1.05)
    - 阴影：增强至 `0 8px 16px rgba(0,0,0,0.1)`
    - 过渡时间：200ms
3. **点击状态**（:active）：
    - 背景：加深10%
    - 缩小：scale(0.98)
    - 按压感：box-shadow减弱

### 6. 进度条更新动画
```javascript
// 数字计数动画
function animateProgress(fromValue, toValue, duration = 400) {
  const startTime = Date.now();
  const startValue = fromValue;
  const difference = toValue - fromValue;
  
  function update() {
    const elapsed = Date.now() - startTime;
    const progress = Math.min(elapsed / duration, 1);
    
    // 使用缓动函数
    const eased = easeOutQuart(progress);
    const currentValue = startValue + difference * eased;
    
    // 更新DOM
    progressElement.textContent = Math.round(currentValue) + '%';
    
    if (progress < 1) {
      requestAnimationFrame(update);
    }
  }
  
  update();
}

// 缓动函数
function easeOutQuart(t) {
  return 1 - Math.pow(1 - t, 4);
}
```

+ **图片压缩**：WebP格式支持，JPEG备选
