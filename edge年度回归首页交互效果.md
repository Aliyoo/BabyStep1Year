## Microsoft Edge 年度回顾页面 - 鼠标晃动交互效果详细描述[1]

### 页面整体结构

该页面采用现代化的视差滚动（Parallax Scrolling）和鼠标跟随（Mouse Tracking）交互设计，创造了富有动感的视觉体验。

### 背景和布局特点

1. **主背景**
   - 采用深色渐变背景，从左上的深蓝色逐渐过渡到右下的深绿色
   - 背景深度分层，营造立体感

2. **核心内容区域**
   - 中央水平居中放置 Microsoft Edge 品牌 logo 和标题文案
   - 标题文案："让我们一起庆祝2025年的特别之处"
   - 下方配置"下一条"操作按钮

3. **浮动卡片元素布局**
   - **左上区域**：蓝色科技感卡片（Edge功能演示）
   - **上中区域**：绿植生活场景卡片
   - **右上区域**：Edge功能截图卡片和橙色花卉装饰
   - **左中区域**：产品功能演示卡片
   - **右中区域**：粉色创意设计卡片
   - **下方区域**：多张生活场景卡片（人物工作、夕阳、家庭生活等）

### 鼠标晃动（跟随）交互效果

当用户移动鼠标在页面上晃动时，会产生以下效果：

1. **视差深度效应**
   - 背景保持相对静止或缓慢移动
   - 浮动的卡片/图片元素以不同速度跟随鼠标移动
   - 距离用户越"近"的元素（z-index 更高）移动幅度越大
   - 距离越"远"的元素（背景层）移动幅度越小

2. **具体表现**
   - 四周的浮动卡片会根据鼠标位置动态调整其偏移量
   - 当鼠标向左移动时，右侧的卡片向右移动，左侧卡片向左移动
   - 当鼠标向上移动时，下方的卡片向下移动，上方卡片向上移动
   - 产生"跟随"的动感视觉，仿佛卡片在追随鼠标光标

3. **动画特性**
   - 动画响应平滑，可能使用了 CSS transform 或 JavaScript requestAnimationFrame 实现
   - 具有一定的缓动效果（easing），使动画看起来流畅而不生硬
   - 可能存在延迟跟随的效果，卡片略滞后于鼠标移动，增强深度感

4. **交互反馈**
   - 鼠标移动越快，卡片的跟随位移越大
   - 鼠标停止移动后，卡片会逐渐返回到默认位置
   - 多个卡片独立响应，各自按照其预设的距离系数进行位移计算

### 设计意图

这种鼠标晃动跟随的交互效果主要用于：
- **增强页面的动感和吸引力**，使静态内容变得生动
- **创造立体感和深度感**，通过视差效应让用户感知到多层次的页面结构
- **提升用户参与度**，鼓励用户与页面互动，增加页面停留时间
- **凸显内容重要性**，通过浮动卡片的突出位置和动画效果强调关键信息
