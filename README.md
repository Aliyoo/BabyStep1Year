# 宝宝的第一年 🍼

一个沉浸式、时间轴型的宝宝成长纪念网站，记录宝宝从 0 到 12 个月的每一个珍贵时刻。

灵感来源于 Microsoft Edge 年度回顾的优美设计风格，通过精心的动画、配色和交互设计，让家长以优雅而富有情感的方式回顾和分享宝宝的成长故事。

## ✨ 功能特性

### 核心功能
- 📅 **13 个月份页面**：从 0 个月到 12 个月（周岁庆典），完整记录宝宝第一年
- 🎨 **独特月份配色**：每个月份拥有专属的渐变背景色，象征不同成长阶段
- 📸 **本地照片上传**：支持 JPG、PNG、WebP 格式，照片存储在浏览器 IndexedDB
- ✏️ **成长故事编辑**：自定义每月的成长故事和里程碑数据
- 💾 **浏览进度记忆**：自动保存浏览位置，下次访问可继续浏览
- 📤 **社交分享**：支持 Facebook、微信、WhatsApp 分享

### 视觉体验
- 🌈 **渐变背景**：45 度斜向渐变，随月份变化
- ✨ **平滑动画**：页面切换、内容淡入淡出、进度计数动画
- 🃏 **毛玻璃卡片**：半透明背景配合 backdrop-filter 效果
- 🖼️ **装饰图片**：两侧装饰图片营造沉浸式氛围
- 📱 **响应式设计**：完美适配桌面、平板、手机

### 🎭 首页视差效果
- 🖱️ **鼠标跟随**：桌面端鼠标移动时，装饰卡片产生深度视差效果
- 📱 **陀螺仪跟随**：移动端支持设备陀螺仪传感器，倾斜设备时卡片跟随移动
- 🖼️ **智能卡片**：优先显示用户上传的月份照片，未上传则显示精美默认宝宝图片
- 🎨 **多肉背景**：柔和的多肉植物背景图，营造温馨自然的氛围
- ⚡ **GPU 加速**：使用 CSS transform3d 和 will-change 优化性能

### 无障碍支持
- 语义化 HTML 标签（main、article、h1 等）
- 完整的 ARIA 属性支持
- 键盘导航支持
- 清晰的焦点轮廓

## 🛠️ 技术栈

- **前端框架**：原生 HTML/CSS/JavaScript（ES Modules）
- **数据存储**：localStorage（进度、文案）+ IndexedDB（照片）
- **路由方案**：Hash-based SPA 路由
- **视差效果**：原生 JavaScript + requestAnimationFrame + CSS transform3d
- **传感器支持**：DeviceOrientation API（移动端陀螺仪）
- **测试框架**：Vitest + jsdom + fake-indexeddb
- **开发服务器**：serve

## 🎮 交互特性

### 桌面端
- 鼠标移动：装饰卡片跟随鼠标产生视差效果
- 鼠标离开：卡片缓慢回到中心位置
- 移动幅度：根据卡片深度值（0.15-0.65）产生不同移动距离

### 移动端
- 设备倾斜：通过陀螺仪感应设备倾斜角度
- 左右倾斜（gamma）：控制卡片 X 轴移动
- 前后倾斜（beta）：控制卡片 Y 轴移动
- 灵敏度：±30° 范围内响应
